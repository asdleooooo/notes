# 外边距重叠

区块的**上下外边距**，有时会发生合并(**大小等于两个编剧中的最大值**)，**设定浮动和绝对定位的元素不会发生外边距折叠**

## 发生折叠的情况

### 相邻兄弟元素

相邻的同级元素之间的外边距会发生折叠，**除非后面的元素需要清楚浮动**
